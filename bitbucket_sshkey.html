🧩 Step-by-step: Add SSH Key for Bitbucket

🧠 Step 1: Check if you already have an SSH key

ls -al ~/.ssh

If you see files like id_ed25519 and id_ed25519.pub, skip to Step 4. If not — continue.

🔑 Step 2: Generate a new SSH key

ssh-keygen -t ed25519 -C "your_email@example.com"

Replace your_email@example.com with your Bitbucket email.

Press Enter for the file location (default is fine):

Enter file in which to save the key (/home/yourname/.ssh/id_ed25519):

Then optionally enter a passphrase or press Enter to skip.

⚙️ Step 3: Start the SSH agent and add your key

eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519

📋 Step 4: Copy your public key

cat ~/.ssh/id_ed25519.pub

Copy the entire output (it starts with ssh-ed25519 and ends with your email).

🌐 Step 5: Add SSH key to Bitbucket

Now open your browser and go to:

https://bitbucket.org/account/settings/ssh-keys/

  • Click “Add key”
  • Paste the copied SSH key into the Key field
  • Give it a label (like “Ubuntu-PC”)
  • Click Add key

✅ Step 6: Test your SSH connection

ssh -T git@bitbucket.org

If everything is set up correctly, you’ll see:

authenticated via ssh key.
You can use git to connect to Bitbucket. Shell access is disabled.

🎉 Done!

You’ve successfully added your SSH key to Bitbucket.
