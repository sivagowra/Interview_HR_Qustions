üß† What is GitHub Actions?

GitHub Actions is a CI/CD tool built into GitHub.

When you push code to GitHub, GitHub can automatically run tasks like build, test, package, create Docker images, and deploy to servers.

No separate Jenkins installation is needed.

üß† One-Line Memory Trick

Code push ‚Üí GitHub Actions runs ‚Üí build/test/deploy automatically

üè† Real-Life Example

Factory Example:

  • Workers = Developers
  • Machine = GitHub Actions
  • Button = Git push

Press button ‚Üí machine works automatically

üîÅ What GitHub Actions Can Do

  • Build Java / Node / Python projects
  • Run tests
  • Create JAR / WAR files
  • Build Docker images
  • Deploy to AWS / EC2 / Kubernetes
  • Run scripts automatically

üìÇ Where GitHub Actions Code is Written

GitHub Actions uses YAML files stored in:

.github/workflows/ci-cd.yml

üìÑ Workflow

A workflow is a set of steps that GitHub runs automatically when certain events occur.

üß© Main Parts of GitHub Actions

Part Meaning
Workflow Full automation process
Job One machine / environment
Step One command inside a job
Action Ready-made reusable task
Runner Server that executes the workflow

üß† Step-by-Step Flow

  1. Developer pushes code
  2. GitHub detects the event
  3. Runner starts (Ubuntu server)
  4. Steps execute one by one
  5. Build created
  6. Result shown in GitHub

‚úçÔ∏è How to Write GitHub Actions

Example 1: Java Build (Maven ‚Üí JAR)

name: Java Build Pipeline
on:
  push:
    branches:
      - main
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - name: Checkout Code
        uses: actions/checkout@v4
      - name: Setup Java
        uses: actions/setup-java@v4
        with:
          java-version: '17'
          distribution: 'temurin'
      - name: Build with Maven
        run: mvn clean package

Example 2: Node.js / React Build

name: Node Build
on: push
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '18'
      - run: npm install
      - run: npm run build

Example 3: Python Script

name: Python Job
on: push
jobs:
  run-python:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: '3.10'
      - run: python app.py

Example 4: Docker Image Build

name: Docker Build
on: push
jobs:
  docker:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: docker build -t myapp:1.0 .
