3Ô∏è‚É£ Sample YAML Pipeline (Python Project)

trigger:
- main

pool:
  vmImage: 'ubuntu-latest'

steps:
- task: UsePythonVersion@0
  inputs:
    versionSpec: '3.x'

- script: |
    pip install -r requirements.txt
    pytest
  displayName: 'Run Tests'

Explain: Triggers on main branch, sets up Python, installs dependencies, runs tests.
