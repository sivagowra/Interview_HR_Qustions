AWS SDK (boto3) – Python Complete Guide

Introduction

AWS SDK (boto3) is the official Python software development kit used to interact with AWS services programmatically. Instead of running AWS CLI commands manually, boto3 allows developers to automate tasks such as creating S3 buckets, launching EC2 instances, uploading files, or managing IAM users through Python code. It is built on top of AWS APIs, making cloud operations faster, more flexible, and easily repeatable inside applications. Boto3 supports both high-level resource APIs and low-level client APIs, giving complete control over AWS services. With boto3, you can integrate AWS into web apps, automate DevOps workflows, process data, or build serverless applications. It also uses your AWS CLI credentials automatically, so once configured, you can run scripts without extra setup. Because it’s pure Python, boto3 works on Linux, Windows, and macOS. In simple terms, AWS SDK (boto3) makes cloud automation easy, powerful, and perfect for developers who want to build scalable AWS-based solutions.

Step 1 – Setup Python Environment

# Create virtual environment
python3 -m venv awsenv

# Activate virtual environment
source awsenv/bin/activate

# Install boto3
pip install boto3

Step 2 – Configure AWS CLI (Credentials)

aws configure

# Enter your credentials:
# AWS Access Key ID → AKIAxxxx
# AWS Secret Access Key → xxxxx
# Default region → ap-south-1
# Output format → json (or leave blank)

boto3 automatically reads credentials from the AWS CLI configuration.

Step 3 – Python Script to List AWS Resources

import boto3

# 1️⃣ S3 – Buckets
s3 = boto3.client('s3')
print("S3 Buckets:")
for bucket in s3.list_buckets()['Buckets']:
    print("-", bucket['Name'])

# 2️⃣ EC2 – Instances
ec2 = boto3.client('ec2')
print("EC2 Instances:")
for reservation in ec2.describe_instances()['Reservations']:
    for instance in reservation['Instances']:
        print("-", instance['InstanceId'], instance['State']['Name'])

# 3️⃣ IAM – Users
iam = boto3.client('iam')
print("IAM Users:")
for user in iam.list_users()['Users']:
    print("-", user['UserName'])

# 4️⃣ Lambda – Functions
lambda_client = boto3.client('lambda')
print("Lambda Functions:")
for fn in lambda_client.list_functions()['Functions']:
    print("-", fn['FunctionName'])

# 5️⃣ RDS – Databases
rds = boto3.client('rds')
print("RDS Databases:")
for db in rds.describe_db_instances()['DBInstances']:
    print("-", db['DBInstanceIdentifier'], db['Engine'], db['DBInstanceStatus'])

# 6️⃣ CloudWatch – Log Groups
logs = boto3.client('logs')
print("CloudWatch Log Groups:")
for log_group in logs.describe_log_groups()['logGroups']:
    print("-", log_group['logGroupName'])

# 7️⃣ SQS – Queues
sqs = boto3.client('sqs')
queues = sqs.list_queues()
print("SQS Queues:")
if 'QueueUrls' in queues:
    for q in queues['QueueUrls']:
        print("-", q)
else:
    print("- No queues found")

# 8️⃣ SNS – Topics
sns = boto3.client('sns')
topics = sns.list_topics()
print("SNS Topics:")
if 'Topics' in topics:
    for topic in topics['Topics']:
        print("-", topic['TopicArn'])
else:
    print("- No topics found")

Step 4 – Run Script

# Activate environment if not active
source awsenv/bin/activate

# Run Python script
python aws_resources.py

The script will print all S3 buckets, EC2 instances, IAM users, Lambda functions, RDS databases, CloudWatch log groups, SQS queues, and SNS topics.

Step 5 – Advantages of Using SDK

  • Automates tasks that are repetitive or large-scale.
  • Access all AWS services programmatically from Python code.
  • Integrates AWS into applications and scripts easily.
  • Works on servers without a GUI.
  • Reduces manual effort and errors.

Step 6 – Disadvantages of Using SDK

  • Beginners may find it difficult to learn commands.
  • Incorrect commands can affect resources (delete/stop).
  • Requires keeping credentials safe.
  • Debugging errors can be tricky for complex scripts.
  • Not as visual as AWS Console dashboards.

10 Simple S3 Examples (boto3)

1. List all buckets

import boto3
s3 = boto3.client("s3")
response = s3.list_buckets()
for b in response["Buckets"]:
    print(b["Name"])

2. Create bucket

import boto3
s3 = boto3.client("s3")
s3.create_bucket(Bucket="my-simple-bucket-12345")
print("Bucket created")

3. Upload file

import boto3
s3 = boto3.client("s3")
s3.upload_file("local.txt", "sivagowra", "file.txt")
print("Uploaded")

4. Download file

import boto3
s3 = boto3.client("s3")
s3.download_file("sivagowra", "file.txt", "downloaded.txt")
print("Downloaded")

5. List objects in a bucket

import boto3
s3 = boto3.client("s3")
response = s3.list_objects_v2(Bucket="sivagowra")
for obj in response.get("Contents", []):
    print(obj["Key"])

6. Delete an object

import boto3
s3 = boto3.client("s3")
s3.delete_object(Bucket="sivagowra", Key="file.txt")
print("Deleted")

7. Copy file inside S3

import boto3
s3 = boto3.client("s3")
source = {"Bucket": "sivagowra", "Key": "file.txt"}
s3.copy(source, "sivagowra", "file_copy.txt")
print("Copied")

8. Generate presigned URL

import boto3
s3 = boto3.client("s3")
url = s3.generate_presigned_url("get_object", Params={"Bucket": "sivagowra", "Key": "file.txt"}, ExpiresIn=300)
print(url)

9. Enable Versioning

import boto3
s3 = boto3.client("s3")
s3.put_bucket_versioning(Bucket="sivagowra", VersioningConfiguration={"Status": "Enabled"})
print("Versioning Enabled")

10. Delete bucket (empty only)

import boto3
s3 = boto3.client("s3")
s3.delete_bucket(Bucket="my-simple-bucket-12345")
print("Bucket Deleted")

10 Simple EC2 Examples (boto3)

1. List all EC2 instances

import boto3

ec2 = boto3.client("ec2")
response = ec2.describe_instances()
for reservation in response['Reservations']:
    for instance in reservation['Instances']:
        print(instance['InstanceId'], instance['State']['Name'])

2. Start an instance

import boto3

ec2 = boto3.client("ec2")
ec2.start_instances(InstanceIds=['i-1234567890abcd'])
print("Instance Started")

3. Stop an instance

import boto3

ec2 = boto3.client("ec2")
ec2.stop_instances(InstanceIds=['i-1234567890abcd'])
print("Instance Stopped")

4. Reboot an instance

import boto3

ec2 = boto3.client("ec2")
ec2.reboot_instances(InstanceIds=['i-1234567890abcd'])
print("Instance Rebooted")

5. Terminate an instance

import boto3

ec2 = boto3.client("ec2")
ec2.terminate_instances(InstanceIds=['i-1234567890abcd'])
print("Instance Terminated")

6. Describe instance status

import boto3

ec2 = boto3.client("ec2")
response = ec2.describe_instance_status()
for status in response['InstanceStatuses']:
    print(status['InstanceId'], status['InstanceState']['Name'])

7. Create a new key pair

import boto3

ec2 = boto3.client("ec2")
key = ec2.create_key_pair(KeyName='my-key-pair')
with open('my-key-pair.pem', 'w') as f:
    f.write(key['KeyMaterial'])
print("Key pair created")

8. Describe security groups

import boto3

ec2 = boto3.client("ec2")
response = ec2.describe_security_groups()
for sg in response['SecurityGroups']:
    print(sg['GroupName'], sg['GroupId'])

9. Create a security group

import boto3

ec2 = boto3.client("ec2")
response = ec2.create_security_group(GroupName='my-sg', Description='My security group')
print("Security Group Created:", response['GroupId'])

10. Add inbound rule to security group

import boto3

ec2 = boto3.client("ec2")
ec2.authorize_security_group_ingress(
    GroupId='sg-12345678',
    IpPermissions=[{
        'IpProtocol': 'tcp',
        'FromPort': 22,
        'ToPort': 22,
        'IpRanges': [{'CidrIp': '0.0.0.0/0'}]
    }]
)
print("Inbound rule added")
