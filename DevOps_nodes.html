What is Cloud?

Cloud (or Cloud Computing) is the delivery of computing services — such as servers, storage, databases, networking, software, and analytics — over the internet instead of using a local computer or physical server. In simple terms: using other companies' computers on the internet to store data or run programs.

☁️ Examples

Store photos on Google Drive
Watch movies on Netflix
Use Gmail or Outlook
Host your website on AWS or Azure

Main Cloud Services

Cloud computing is usually divided into three main types of services:

Type Full Form Description Example
IaaS Infrastructure as a Service Provides virtual machines, servers, and storage. AWS EC2, Google Compute Engine
PaaS Platform as a Service Provides development platforms and tools to build apps. AWS Elastic Beanstalk, Google App Engine
SaaS Software as a Service Provides ready-to-use applications over the internet. Gmail, Zoom, Salesforce

Benefits of Cloud Computing

Cost-Effective
No need to buy hardware.
Scalable
Increase or decrease resources anytime.
Secure
Data protected by encryption and backups.
Accessible Anywhere
Only need an internet connection.
Automatic Updates
Providers manage updates.
