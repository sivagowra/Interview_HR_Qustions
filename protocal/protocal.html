PORT NUMBERS FOR EVERY DEVOPS ENGINEER

Essential Port Numbers for DevOps Engineers

Common Protocols and Services

Service Port Purpose
HTTP 80 Unsecured web traffic.
HTTPS 443 Secure web traffic (TLS/SSL).
SSH 22 Secure shell access for remote administration.
FTP 20 (data transfer), 21 (control) File Transfer Protocol.
SFTP 22 Secure FTP via SSH.
Telnet 23 Unsecured remote administration (rarely used now).
DNS 53 Domain Name System resolution.
SMTP 25, 587 (with encryption), 465 (deprecated) Email sending.
IMAP 143 (unencrypted), 993 (encrypted) Retrieve emails (modern email clients).
POP3 110 (unencrypted), 995 (encrypted) Retrieve emails (older protocol).

DevOps and Automation Tools

Tool/Service Port Purpose
Jenkins 8080 (default) Continuous Integration/Delivery.
Docker 2375 (unencrypted), 2376 (encrypted) Docker API.
Kubernetes API Server 6443 Communication with Kubernetes API.
Terraform Enterprise 8800 (default) Backend services.

Databases

Database Port Purpose
MySQL 3306 MySQL database.
PostgreSQL 5432 PostgreSQL database.
MongoDB 27017 MongoDB NoSQL database.
Redis 6379 In-memory data store.
Elasticsearch 9200 (HTTP), 9300 (Node communication) Search and analytics engine.

Cloud and Monitoring Tools

Tool/Service Port Purpose
Prometheus 9090 Monitoring and alerting.
Grafana 3000 Data visualization and dashboards.
Consul 8500 (HTTP API), 8600 (DNS queries) Service discovery and configuration.
Vault 8200 Secrets management.
Zabbix 10051 (server), 10050 (agent) Monitoring.

Version Control

Protocol Port Purpose
Git (SSH) 22 Secure Git operations.
Git (HTTPS) 443 Secure Git operations via HTTPS.

Messaging and Queueing Systems

System Port Purpose
RabbitMQ 5672 (AMQP), 15672 (Management console) Message broker.
Kafka 9092 Distributed messaging system.

Miscellaneous

Protocol/Service Port Purpose
NTP 123 Network Time Protocol.
LDAP 389 (unencrypted), 636 (encrypted) Directory services.
SNMP 161 (queries), 162 (traps) Network monitoring.
RDP 3389 Remote Desktop Protocol (Windows).
