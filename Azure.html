AWS to Azure Services Mapping

AWS Service Purpose Azure Equivalent
EC2 Virtual machines for compute Azure Virtual Machines (VMs)
S3 Object storage service Azure Blob Storage
VPC Virtual private cloud network Azure Virtual Network (VNet)
SQS Message queue service Azure Queue Storage / Azure Service Bus Queues
SNS Notification + pub/sub messaging Azure Event Grid / Azure Notification Hubs / Service Bus Topics
CloudWatch Monitoring & logging Azure Monitor + Azure Log Analytics
IAM Identity & access management Azure Active Directory (Entra ID)
RDS Managed relational database Azure SQL Database / Azure Database for MySQL/PostgreSQL
